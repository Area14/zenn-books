## Claude Designとは

2026年4月、Anthropicが発表した「Claude Design」は、テキスト対話だけでUIプロトタイプ、プレゼンテーション、インタラクティブデモ、ホワイトボードスケッチを生成できるAIデザインツールです。

Figma株が発表直後に4%以上下落するほどのインパクトがありました。

## なぜ本を書いたのか

公式ヘルプは概要レベルにとどまっています。特に以下の領域は体系的な解説がまだありませんでした:

- **デザインシステムの構築手順** — アセットのアップロードから自動抽出、レビュー、チーム公開まで
- **Claude Codeとの連携ワークフロー** — デザインからハンドオフ、実装までの一気通貫フロー
- **各モードの使い分け** — Prototype / Slide deck / Code Prototype / Sketch の最適な活用シーン

## 本書のポイント

### 17本のハンズオン

全12章を通じて17本のハンズオンを収録。読みながら手を動かすことで確実にスキルが身につきます。

### デザインシステム構築をゼロから

Claude Designで最も強力な機能がデザインシステムです。本書ではブランドアセットのアップロードから自動抽出、Publishまでをステップバイステップで解説しています。

### Claude Code連携

デザインをClaude Codeにハンドオフして実装まで進める方法もカバー。デザイナーとエンジニアの分業を超えた、AIネイティブな開発フローを紹介します。

## 対象読者

- デザインツール経験が少ないエンジニア・PM
- 個人開発でUIやプレゼンの品質を上げたい方
- Claude Codeは使っているがDesign機能はまだの方

デザインの専門知識は不要。Claudeの基本操作を知っていれば読み進められます。
