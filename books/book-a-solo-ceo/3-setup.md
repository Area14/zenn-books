## 準備するもの一覧

AIオフィスの開設に必要なものを確認します。

| 必要なもの | 費用 | 備考 |
|:---|:---|:---|
| パソコン（Windows or Mac） | — | すでにお持ちのもので十分 |
| インターネット環境 | — | 普通の速度で問題なし |
| メールアドレス | 無料 | Claude アカウント用 |
| Claude Pro または Max プラン | 月額 $20〜 | Claude Code 機能が必要 |
| Slack アカウント | 無料 | フリープランでOK |
| GitHub アカウント | 無料 | Claude Code in Slack の連携に必要 |

「GitHub（ギットハブ）」という聞き慣れない名前が出てきましたね。これは本来プログラマーがコード（プログラム）を管理するためのサービスですが、Claude Code in Slackを使うためには、アカウントだけ作っておく必要があります。プログラミングの知識は不要です。アカウントを作るだけで結構です。

それでは、ひとつずつセットアップしていきます。


## ハンズオン① Claudeアカウントを作成する

まずは、AI本体であるClaudeのアカウントを作ります。

**手順：**

1. パソコンのブラウザ（Chrome、Safari、Edgeなど）で `claude.ai` にアクセスします
2. 「Sign up」（サインアップ）をクリックします
3. メールアドレスを入力して「Continue」をクリックします
   - Googleアカウントでのサインアップも可能です
4. 届いた確認メールのリンクをクリックします
5. 名前を入力して、利用規約に同意します

これでClaudeの無料アカウントが作成されました。

**次に、有料プランにアップグレードします。**

Claude Code in Slackの機能を使うには、Pro（プロ）プラン以上が必要です。

6. 画面左下の「Upgrade」または歯車アイコン → 「Subscription」をクリック
7. プランを選択します

### プランの選び方

| プラン | 月額（税別） | おすすめの人 |
|:---|:---|:---|
| Pro | $20（約3,000円） | まず試してみたい人。月の利用量に上限あり |
| Max 5x | $100（約15,000円） | 日常的にAIエージェントを使いたい人。利用量5倍 |
| Max 20x | $200（約30,000円） | ヘビーに使い倒したい人。利用量20倍 |

**筆者のおすすめはPro（月$20）からスタート**することです。本書のハンズオンをひと通り試すには十分な利用量です。使い方に慣れて「もっと使いたい」と思ったら、いつでもMax 5xにアップグレードできます。

8. クレジットカード情報を入力して、プランの購入を完了します

:::message
**ポイント**
料金はUSドル建てです。為替レートによって日本円の金額は変動します。月額$20であれば、外注でちょっとした資料作成を1回頼む程度の金額です。AIチーム5人分と考えれば、破格のコストパフォーマンスです。
:::


## ハンズオン② GitHubアカウントを作成する

次に、GitHubのアカウントを作ります。繰り返しますが、プログラミングの知識は不要です。

**手順：**

1. ブラウザで `github.com` にアクセスします
2. 「Sign up」をクリックします
3. メールアドレス、パスワード、ユーザー名を入力します
   - ユーザー名は英数字で、あとで変更も可能です
4. 届いた確認メールのリンクをクリックします
5. 無料プラン（Free）のまま「Continue」で進みます

これだけです。GitHubのダッシュボード（管理画面）が表示されたら完了です。このあと、Claude Code on the Webとの連携で使います。


## ハンズオン③ SlackにClaude Appをインストール・連携する

いよいよSlackの設定です。Claude Appのインストールから、アカウント連携、Claude Code on the Webの設定まで、一気に進めます。すでにSlackを使っている方は、そのワークスペースに追加できます。Slackを使ったことがない方は、まずSlackアカウントの作成から始めてください。

### Slackが初めての方

1. `slack.com` にアクセス
2. 「始める」をクリック
3. メールアドレスを入力してワークスペースを作成
4. ワークスペース名は「My AI Office」など、お好みで

### ステップ1：Claude Appのインストール

**手順：**

1. Slackを開きます（ブラウザ版でもデスクトップアプリでもOK）
2. ブラウザの別タブで、Slack App Marketplace の Claude App ページにアクセスします
   - Slackの左サイドバーから「App」→「Appを検索」→「Claude」で検索してもOK
3. 「Add to Slack」（Slackに追加）をクリック
4. アクセス権限の確認画面が出ます。「Allow」（許可）をクリック
5. Slackに戻ると、左サイドバーの「App」セクションに「Claude」が追加されています

:::message alert
**注意**
ワークスペースの管理者権限が必要です。個人で作ったワークスペースなら、あなたが管理者なので問題ありません。会社のSlackに追加する場合は、IT管理者の許可が必要です。
:::

### ステップ2：Claudeアカウントを連携する

Claude AppをSlackにインストールしただけでは、まだあなた個人のClaudeアカウントとは繋がっていません。連携が必要です。

**手順：**

1. Slackの左サイドバーで「Claude」アプリをクリックします
2. 「App Home」タブが表示されます
3. 「Connect」ボタンをクリックします
4. ブラウザが開き、Claudeのログイン画面が表示されます
5. さきほど作成したClaudeアカウントでログインします
6. 連携の許可を求められるので「Allow」をクリック
7. Slackに戻ると、「Connected」（接続済み）と表示されます

これで、あなたのClaudeアカウントとSlackが繋がりました。

### ステップ3：Claude Code on the Webの設定

Claude Code in Slackの裏側では、「Claude Code on the Web」という機能が動いています。ここにGitHubアカウントを接続し、少なくとも1つのリポジトリを認証しておく必要があります。

**手順：**

1. ブラウザで `claude.ai/code` にアクセスします
2. Claudeアカウントでログインします（まだの場合）
3. 画面の案内に従って、GitHubアカウントを接続します
   - 「Connect GitHub」のようなボタンが表示されます
   - GitHubのログインを求められるので、さきほど作ったアカウントでログイン
   - リポジトリ（保管場所）へのアクセス許可を求められます。「All repositories」を選んで問題ありません
4. **少なくとも1つのリポジトリが認証済みになっていることを確認します**
   - 接続したGitHubアカウントにリポジトリがない場合は、GitHub上で新しいリポジトリを1つ作成してください（名前は「ai-office」などで構いません）
5. 接続が完了すると、Claude Code on the Webの画面が表示されます

:::message
**ポイント**
「リポジトリ」というのは、ファイルの保管場所のことです。AIエージェントが作業するとき、作成したファイルはこのリポジトリに保存されます。最初は空で問題ありません。ただし、少なくとも1つのリポジトリが認証されていないと、Slackからのセッションが開始されません。
:::


## ハンズオン④ ルーティングモード設定と5つの部署チャンネル作成

Claudeがメッセージをどう処理するかの設定と、AIオフィスの部署チャンネルを一気に作ります。

### ステップ1：ルーティングモードの設定

**手順：**

1. Slackに戻り、左サイドバーの「Claude」アプリをクリック
2. 「App Home」タブを開きます
3. **Routing Mode**（ルーティングモード）の設定を探します
4. **「Code + Chat」**を選択します

### ルーティングモードとは

| モード | 動作 |
|:---|:---|
| Code only | すべてのメッセージをClaude Code（作業モード）で処理する |
| **Code + Chat**（推奨） | メッセージの内容を判断して、作業系はClaude Code、雑談や質問はチャットで返答する |

「Code + Chat」がおすすめです。このモードなら、「おはよう、今日のタスクは？」のような質問にはチャットで返答し、「売上データを集計してレポートにして」のような作業依頼にはClaude Codeが起動します。

もし判断が間違っていた場合（チャットで返答されたけど、本当は作業してほしかった場合）は、「Retry as Code」ボタンをクリックすれば、Claude Codeモードでやり直してくれます。

### ステップ2：5つの部署チャンネルを作成する

セットアップの仕上げです。第2章で設計した5つの部署チャンネルをSlackに作ります。

**手順：**

1. Slackの左サイドバーで「チャンネル」の横にある「＋」をクリック
2. 「チャンネルを作成する」を選択
3. 以下の5つのチャンネルを順番に作成します

| チャンネル名 | 説明（Description） |
|:---|:---|
| `ai-secretary` | AI秘書：スケジュール、議事録、メール下書き |
| `ai-research` | AIリサーチャー：市場調査、競合分析、トレンド |
| `ai-accounting` | AI経理：経費集計、売上管理、月次レポート |
| `ai-sales` | AI営業：提案書、顧客管理、フォローメール |
| `ai-marketing` | AIマーケ：SNS投稿、ブログ、広告コピー |

4. 各チャンネルの作成時に、公開（Public）か非公開（Private）を選べます
   - ひとり社長の場合は、どちらでも構いません
   - セキュリティを重視するなら非公開がおすすめです

5. **各チャンネルにClaudeを招待します**
   - チャンネルに入ったら、メッセージ欄に `/invite @Claude` と入力してEnter
   - 「Claude has been added to #ai-secretary」のようなメッセージが出ればOK
   - 5つすべてのチャンネルで繰り返します

:::message alert
**重要**
`/invite @Claude` を忘れると、そのチャンネルでは@Claudeと話しかけてもエージェントが反応しません。全チャンネルで確実に招待してください。
:::

## トラブル対処：セッションが始まらないときの確認ポイント

セットアップ後に「@Claudeに話しかけたのに反応がない」という場合は、以下を確認してください。

**1. チャンネルにClaudeが招待されているか**
→ チャンネルのメンバー一覧に「Claude」がいるか確認。いなければ `/invite @Claude` で招待。

**2. Claudeアカウントが連携されているか**
→ Slackの「Claude」アプリ → App Home タブで「Connected」と表示されているか確認。

**3. Claude Code on the Webが有効か**
→ `claude.ai/code` にアクセスして、正常に表示されるか確認。

**4. GitHubアカウントが接続されているか**
→ Claude Code on the Web の設定画面で、GitHubが接続済みになっているか確認。

**5. プランがPro以上か**
→ Claudeの設定画面でプランを確認。Free（無料）プランではClaude Code機能が使えません。

**6. DM（ダイレクトメッセージ）で話しかけていないか**
→ Claude Code in Slackの作業機能（コード実行やファイル操作）は、チャンネル内でのみ動作します。DMでは通常のチャット応答のみ利用可能で、Claude Codeセッションは起動しません。必ずチャンネル内で@Claudeをメンションしてください。

これらを確認しても解決しない場合は、SlackのClaude Appを一度削除して再インストールするか、Claudeアカウントの連携を解除して再接続してみてください。
