## 経営者に最も近い存在

5人のAIエージェントの中で、最初に配属するのはAI秘書です。

なぜ秘書からか。理由はシンプルです。秘書は「何でも屋」であり、経営者の日常業務で最も広い範囲をカバーするからです。議事録の作成、メールの下書き、タスクの整理、資料のアウトライン作成。種類はバラバラでも、ひとつひとつは比較的シンプルな仕事が多い。つまり、AIエージェントの使い方を覚えるのに最適な「練習相手」でもあるのです。

この章を終える頃には、あなたは毎朝 `#ai-secretary` チャンネルを開いて、AI秘書と今日の段取りを確認するのが習慣になっているはずです。


## AI秘書の担当業務

まず、AI秘書にどんな仕事を任せるのかを整理します。

| 業務カテゴリ | 具体的なタスク例 |
|:---|:---|
| 議事録・記録 | 会議メモの整形、議事録の作成、決定事項の抽出 |
| メール・文書 | メールの下書き、お礼状、お詫びの文面、契約書の確認依頼文 |
| タスク管理 | 今週のToDoリスト整理、優先順位の提案、締め切りリマインド |
| 情報整理 | 散在するメモの統合、会議用資料の要点まとめ |
| スケジュール | 会議のアジェンダ作成、1週間の予定サマリー |

すべてを一度にやろうとする必要はありません。まずはひとつ、最も時間を食っている業務から試してみてください。


## 秘書エージェントの「役割設定」を作る

これまでは `#ai-secretary` チャンネルで@Claudeに話しかけるだけでしたが、**毎回「あなたは秘書です」と伝えるのは面倒**ですよね。そこで、「ロールプロンプト」（役割設定）という仕組みを使います。

### ロールプロンプトとは

ロールプロンプトとは、AIエージェントに「あなたはこういう役割です。こういうルールで仕事をしてください」と伝えるテキストのことです。

人間の新入社員に例えるなら、入社初日に渡す「業務マニュアル」のようなものです。

- あなたの肩書きは「秘書」です
- 話し方は丁寧語を使ってください
- 報告は箇条書きで、簡潔にまとめてください
- 判断に迷ったら「確認が必要です」と言ってください

このような指示を事前に用意しておけば、毎回のメッセージで細かい指定をしなくても、最初から秘書としての振る舞いをしてくれます。

### Slackでの運用方法：「ピン留め」で役割を固定する

秘書エージェントの役割設定を、Slackのチャンネルに「ピン留め」しておきます。こうすることで、いつでもコピペで使えるようになります。

**手順：**

1. `#ai-secretary` チャンネルを開きます
2. 以下のロールプロンプトをメッセージとして投稿します（@Claudeのメンションは**つけない**）

```
【AI秘書 ロールプロンプト】
以下をコピーして@Claudeへのメッセージの冒頭に貼り付けてください。

---
あなたは、ひとり社長の専属秘書AIアシスタントです。

■ 役割
- 経営者の日常業務を効率化するためのサポートを行います
- 議事録作成、メール下書き、タスク整理、資料準備が主な業務です

■ 対応方針
- 常に丁寧語（です・ます調）で対応してください
- 報告は簡潔に、箇条書きを活用してください
- 重要度の高い項目には【重要】マークを付けてください
- 判断が必要な事項は、選択肢を提示した上で社長の判断を仰いでください
- 「確認事項」があれば、レポートの最後にまとめてください

■ 出力形式
- タスクリスト：番号付きリスト、期限と優先度を付記
- 議事録：日時・参加者・議題・決定事項・次のアクションの形式
- メール下書き：件名・本文・署名の形式
- レポート：見出し → 要点 → 詳細の階層構造

■ 注意事項
- 社長の時間は貴重です。冗長な説明は避け、結論ファーストで報告してください
- 不明点がある場合は推測で進めず、確認してください
---
```

3. 投稿したメッセージにカーソルを合わせ、「...」（その他）→「ピン留めする」をクリック

これで、チャンネルの上部にある「ピン留めアイテム」からいつでもこのロールプロンプトにアクセスできます。

### 日常的な使い方

毎回全文を貼り付ける必要はありません。Claudeはスレッド内の文脈を覚えるので、**最初のメッセージだけ**ロールプロンプトをつければOKです。

```
@Claude 以下の役割で対応してください。
（ピン留めからロールプロンプトを貼り付け）

今日の打ち合わせのアジェンダ案を作って。
議題は以下の3つ：
1. 新サービスの進捗確認
2. 来月の販促計画
3. オフィス移転の検討
```

2回目以降の同じスレッド内では、ロールプロンプトなしで追加指示を出せます。

:::message
**ポイント**
使い込むうちに「もう少し簡潔にしてほしい」「表をもっと使ってほしい」といった好みが見えてきます。そのたびにピン留めのロールプロンプトを更新していくのがおすすめです。最初から完璧を目指す必要はありません。
:::

もっとシンプルに使いたい場合は、短い一文でも十分です。

```
@Claude （秘書として対応して）今日の打ち合わせのアジェンダ案を作って。
```

この「（秘書として対応して）」のひと言を添えるだけで、Claudeの応答のトーンと形式が変わります。


## ハンズオン⑦ 会議メモから議事録を自動生成する

では、秘書エージェントの実力を試してみます。最も使用頻度が高い「議事録作成」です。

**`#ai-secretary` チャンネルで、以下のメッセージを送信してください：**

```
@Claude 以下の会議メモから、正式な議事録を作成してください。

---
4/10 14時〜 オンラインMTG
参加：自分、田中さん（クライアント）、佐藤さん（クライアント側デザイナー）

・新コーポレートサイトのリニューアル案件
・田中さん「5月末までにはリリースしたい」
・トップページのデザインは3案出す方向で合意
・佐藤さん「素材の写真は来週中にDropboxで共有する」
・予算は150万〜200万の範囲
・次回MTGは4/17 14時、デザイン初稿のレビュー
・田中さん「社長にも見せたいので、プレゼン用のモックアップも欲しい」
---

議事録の形式：
- 日時・場所
- 参加者
- 議題
- 決定事項（箇条書き）
- 各自のアクションアイテム（担当者・期限つき）
- 次回会議の情報
```

数秒で、きれいに整形された議事録が返ってきます。

走り書きのメモが、クライアントに共有しても恥ずかしくない正式な議事録に変わる。この変換にかかる時間は、人間が手作業でやれば15〜30分。AI秘書なら10秒です。


## ハンズオン⑧ 毎朝のタスク整理を習慣にする

次は、AI秘書を活用した**朝のルーティン**を作ります。毎朝Slackを開いて、決まったプロンプトを送信するだけ。1分でその日の段取りが整います。

### 「朝の定型プロンプト」をピン留めする

まず、毎朝使うプロンプトを `#ai-secretary` チャンネルにピン留めしておきます。

**ピン留めする内容：**

```
【毎朝のタスク整理プロンプト】
以下をコピーして @Claude に送信してください。

---
@Claude おはようございます。本日のタスク整理をお願いします。

以下のフォーマットで出力してください：

## 本日のタスク整理（今日の日付と曜日を入れて）

### 最優先（今日中に必ず）
-（あれば記載、なければ「なし」）

### 重要（今週中）
-

### その他
-

### 今日のリマインド
-（締め切りが近いもの、返信待ちのものなど）

### ひとこと
-（前向きになれるひとことを添えて）
---
```

### 朝のルーティンの流れ

1. 朝、Slackを開く
2. `#ai-secretary` チャンネルのピン留めから、タスク整理プロンプトをコピー
3. メッセージ欄に貼り付けて送信
4. 数秒でタスク整理が返ってくる
5. 内容を確認して、今日の優先順位を決める

所要時間は1分。コーヒーを淹れている間に終わります。

:::message
**もっと便利に使うコツ**
前日の終わりに「明日やること」をスレッドで伝えておくと、翌朝のタスク整理に反映されます。退勤前に `@Claude 明日のToDoとして、A社への見積書作成とB社へのフォローメールを覚えておいて` と送信しておくだけです。
:::

### 将来的な自動化

「毎朝自分で送るのではなく、自動的にメッセージが届いてほしい」と思うかもしれません。Claude Managed Agentsのスケジュール実行機能は現在進化中で、今後、時間指定でエージェントを自動起動する機能がより使いやすくなることが予告されています。本書の執筆時点（2026年4月）では、上記の「手動ルーティン」方式が最も確実で簡単な方法です。

朝、コーヒーを淹れてSlackを開き、ワンタップでタスク整理を依頼する。まるで、デスクに座ったら秘書がその日の資料を揃えてくれるような感覚です。


## 秘書エージェント活用のコツ

1ヶ月ほどAI秘書を使ってみると、いくつかのコツが見えてきます。先にお伝えしておきます。

### コツ1：テンプレートを育てる

議事録、メール下書き、タスクリスト。よく使う出力形式は、「自分好みのテンプレート」に育てていくのがポイントです。最初にClaudeが返してきたフォーマットが気に入ったら、それを「次からもこの形式で」と指定します。

```
@Claude 議事録を作成して。フォーマットは前回と同じで。
---
（会議メモをここに貼り付け）
---
```

### コツ2：「判断」は任せない、「作業」を任せる

AI秘書が得意なのは、「情報の整理」と「テキストの生成」です。一方、「この見積もり金額は妥当か」「このクライアントとの契約を継続すべきか」といった経営判断は、必ずあなた自身が行ってください。

AIは情報を集めて整理するのが仕事。判断するのが社長の仕事。この線引きを忘れないでください。

### コツ3：フィードバックを繰り返す

「もう少し簡潔にして」「箇条書きをもっと使って」「結論を最初に持ってきて」——こうしたフィードバックをスレッドで伝えるほど、その会話内でのAI秘書の出力品質が上がります。

人間の秘書も、長く一緒に働くほど「社長の好み」を覚えていきますよね。AIエージェントも同じです。ただし、AIの場合は「セッション」が変わると忘れてしまうため、重要な好みはロールプロンプト（ピン留め）に追記しておくのがベストです。


## テンプレート：秘書エージェント ロールプロンプト カスタマイズ例

本章で紹介したロールプロンプトは基本形です。使い込むうちに、自分の業種や好みに合わせてカスタマイズしていくことをおすすめします。

以下に、業種別のカスタマイズ例を示します。

**コンサルタントの場合：追加する一文**
```
- クライアントへの報告文書は、経営層が読むことを想定し、専門用語は極力避けてください
- 数値には必ず比較対象（前月比、前年比など）を添えてください
```

**ECショップ運営者の場合：追加する一文**
```
- 商品名、SKU、在庫数などの数値情報は正確性が最優先です
- 不確かな数値は「要確認」と明記してください
```

**クリエイター（デザイナー・ライター）の場合：追加する一文**
```
- 制作スケジュールは、修正回数を2回見込んだバッファを含めてください
- クライアントへの連絡文は、クリエイティブな印象を大切にしつつも、ビジネスライクな表現で
```

自分にフィットするロールプロンプトが出来上がったら、それはあなただけの「秘書の教育マニュアル」です。大切に育てていってください。


## 次章の予告

AI秘書が動き始めると、最初に感じるのは「もっといろんなことを頼みたい」という欲求です。特に、ひとり社長が最も時間を取られる「調べ物」を誰かに任せたくなります。

次の章では、`#ai-research` チャンネルにAIリサーチャーを配属します。競合調査、市場分析、トレンド把握。あなたが「ちょっと調べて」と言えば、数分で構造化されたレポートが返ってくる世界を体験してください。
