## 数字が苦手でも、AIが計算してくれる

「数字は苦手で……」

この言葉を、何人のひとり社長から聞いたか分かりません。デザインやコンサルティングなど、クリエイティブな仕事で独立した方にとって、経理業務はまさに「やりたくないけど、やらなければならない」仕事の筆頭です。

領収書の整理、経費の仕分け、請求書の発行と入金確認、月次の収支報告。税理士にお願いしているとしても、元データの整理はこちらの仕事です。

AI経理が「面倒な数字の仕事」をどこまで引き受けてくれるか、実際に手を動かしながら確かめます。結論を先に言えば、**あなたが数字に強い必要はない。CSVファイルを渡して「集計して」と言えばいい**のです。


## AI経理の担当業務

| 業務 | AIにできること |
|:---|:---|
| 経費集計 | CSVデータの分類・集計・グラフ化 |
| 売上管理 | 月別・顧客別・案件別の売上集計 |
| 月次レポート | 収支サマリーの自動生成 |
| 請求管理 | 請求一覧の整理、未入金アラート |
| 予実管理 | 予算と実績の比較表作成 |

ただし、AI経理は「計算と整理」をするのであって、「税務判断」はしません。税金の申告や節税のアドバイスは、必ず税理士に相談が必要です。AI経理の守備範囲は「データの下処理と見える化」に限られます。


## CSVファイルの扱い方

AI経理に仕事を頼むとき、最も基本的なインプット（入力データ）になるのが「CSVファイル」です。

### CSVファイルとは

CSV（Comma-Separated Values）とは、表のデータをテキスト形式で保存したファイルです。Excelやスプレッドシートで作った表を、CSVとして書き出す（エクスポートする）ことができます。

たとえば、こんな売上データがあるとします。

| 日付 | 顧客名 | 案件名 | 金額 |
|:---|:---|:---|---:|
| 2026/01/15 | A社 | コーポレートサイト制作 | 800,000 |
| 2026/02/01 | B社 | LP制作 | 300,000 |
| 2026/02/20 | C社 | ECサイトリニューアル | 1,200,000 |

これをCSVにすると、こうなります。

```
日付,顧客名,案件名,金額
2026/01/15,A社,コーポレートサイト制作,800000
2026/02/01,B社,LP制作,300000
2026/02/20,C社,ECサイトリニューアル,1200000
```

### CSVの作り方

会計ソフト、Excelやスプレッドシート、あるいはクラウドサービスから、データをCSV形式で書き出すだけです。

- **Excel**：ファイル → 名前を付けて保存 → ファイル形式を「CSV（コンマ区切り）」に変更
- **Googleスプレッドシート**：ファイル → ダウンロード → カンマ区切り値（.csv）
- **freee、マネーフォワード等**：各種レポート画面の「CSVダウンロード」ボタン

### SlackにCSVを渡す方法

Slackのメッセージ欄にCSVファイルをドラッグ＆ドロップ（マウスで引っ張って放す）するだけです。あるいは、メッセージ欄の「＋」アイコンから「ファイルをアップロード」を選択することもできます。

ファイルを添付した上で、@Claudeへのメッセージを書いて送信します。


## Pythonスクリプトの自動生成について

AI経理にデータ集計を頼むと、ほとんどの場合、裏側で「Python（パイソン）」というプログラミング言語のスクリプト（プログラム）を自動生成して実行します。

**「プログラミングなんて分からない」と思った方も、問題ありません。**

AIが自動でプログラムを書いて、自動で実行して、結果だけを返してくれます。あなたはプログラムの中身を理解する必要は一切ありません。レストランで料理を注文するとき、シェフの調理手順を知らなくても美味しい料理が出てくるのと同じです。

Claude Managed Agentsのサンドボックス（安全な作業環境）には、あらかじめPythonとデータ分析用のライブラリがインストールされています。あなたが事前に何か準備する必要は一切ありません。


## ハンズオン⑪ 売上CSVから月次サマリーを自動生成する

では、実際に手を動かします。

まず、サンプルの売上データを作ります。お手持ちのデータがあれば、それを使っていただいても構いません。

**ステップ1：サンプルCSVを作成する**

Excelやメモ帳で以下の内容を `sales_2026.csv` として保存してください。

```
日付,顧客名,案件名,カテゴリ,金額,ステータス
2026/01/15,A社,コーポレートサイト制作,Web制作,800000,入金済
2026/01/28,B社,ロゴデザイン,デザイン,150000,入金済
2026/02/01,C社,LP制作,Web制作,300000,入金済
2026/02/15,D社,ECサイトリニューアル,Web制作,1200000,入金済
2026/02/20,A社,バナーデザイン5点,デザイン,100000,入金済
2026/03/01,E社,コーポレートサイト制作,Web制作,600000,請求中
2026/03/10,B社,名刺デザイン,デザイン,50000,入金済
2026/03/15,F社,Webコンサルティング,コンサル,200000,請求中
2026/03/25,C社,サイト保守（3月分）,保守,80000,入金済
2026/04/01,G社,LP制作,Web制作,350000,未請求
```

**ステップ2：Slackでデータを送る**

`#ai-accounting` チャンネルを開き、上記CSVファイルをドラッグ＆ドロップで添付し、以下のメッセージを送信します。

```
@Claude 添付の売上データ（sales_2026.csv）を分析して、月次サマリーレポートを作成してください。

レポートに含めてほしい内容：
1. 月別の売上合計（表）
2. カテゴリ別の売上構成比（表）
3. 顧客別の売上ランキング
4. 入金ステータス別の集計（入金済 / 請求中 / 未請求）
5. 前月比の増減

出力形式：
- 各セクションに見出しをつけて
- 数値にはカンマ区切りで円マークをつけて（例：¥1,200,000）
- 最後に「経営者向けの所見」を3行程度で
```

Claude Codeセッションが起動し、以下のことが自動で行われます。

1. CSVファイルを読み込む
2. Pythonでデータを集計するプログラムを自動生成・実行
3. 集計結果を分かりやすいレポート形式にまとめる
4. Slackに結果を投稿

数分後、きれいに整形された月次サマリーが返ってきます。


## ハンズオン⑫ 経費CSVを分類して可視化する

売上の次は、経費の集計です。

**サンプルの経費データ `expenses_2026.csv`**（金額は概算です）**：**

```
日付,内容,金額,支払方法
2026/01/05,レンタルサーバー（年額）,15000,クレジットカード
2026/01/10,Adobe Creative Cloud,6480,クレジットカード
2026/01/15,コワーキングスペース月額,22000,口座引落
2026/01/20,書籍「Webデザインの教科書」,2800,クレジットカード
2026/02/01,Zoom Pro月額,2000,クレジットカード
2026/02/10,Adobe Creative Cloud,6480,クレジットカード
2026/02/15,コワーキングスペース月額,22000,口座引落
2026/02/20,交通費（クライアント訪問）,1200,現金
2026/02/25,文房具,500,現金
2026/03/05,ドメイン更新,1500,クレジットカード
2026/03/10,Adobe Creative Cloud,6480,クレジットカード
2026/03/12,名刺印刷100枚,3000,クレジットカード
2026/03/15,コワーキングスペース月額,22000,口座引落
2026/03/18,セミナー参加費,5000,クレジットカード
2026/03/28,交通費（セミナー会場）,800,現金
```

**Slackで送るメッセージ：**

```
@Claude 添付の経費データ（expenses_2026.csv）を分析してください。

やってほしいこと：
1. 各経費を以下のカテゴリに自動分類してください
   - 通信費（サーバー、ドメイン、Zoom等）
   - ソフトウェア費（Adobe等）
   - 地代家賃（コワーキング）
   - 旅費交通費
   - 研修費（セミナー、書籍）
   - 消耗品費（文房具、名刺）

2. 以下のレポートを作成してください
   - カテゴリ別の経費集計表（月別）
   - 総経費の月別推移
   - 支払方法別の集計
   - 最も支出が大きいカテゴリのハイライト

3. 可能であれば、棒グラフまたは円グラフも生成してください

出力形式：表とグラフ、最後に節約のヒントを1〜2行
```

ここが面白いところです。AIは経費の「内容」の文言を読み取って、自動的にカテゴリに分類してくれます。「Adobe Creative Cloud → ソフトウェア費」「コワーキングスペース → 地代家賃」といった判断をAIが行うのです。

さらに、Pythonのグラフ描画ライブラリを使って、視覚的なグラフまで自動生成してくれます。


## 機密データの取り扱いとセキュリティの基本

経理データは、会社の重要な機密情報です。AIエージェントに渡すデータの取り扱いについて、いくつか重要なポイントを押さえておく必要があります。

### Claude Code in Slackのセキュリティ

Anthropicが提供するClaude Codeの基盤には、以下のセキュリティ対策が組み込まれています。

- **隔離された実行環境**：あなたのデータは、他のユーザーのデータと分離されたクラウドコンテナ（隔離された仮想空間）で処理されます
- **認証情報の保護**：パスワードやAPIキーなどの機密情報は、作業環境とは別の安全な領域（Vault）で管理されます
- **セッション管理**：セッションが終了すると、実行環境はクリーンアップされます。ただし、作成されたファイルはGitHubリポジトリに保存されるため、必要に応じて後から確認できます

### それでも気をつけるべきこと

1. **個人の銀行口座番号やクレジットカード番号は送らない**
   - 集計に必要な「金額」は送っても、「口座番号」は不要です
   - 不要な個人情報は、CSVから削除してから送ります

2. **顧客の個人情報は最小限にする**
   - 集計のためなら「A社」「B社」と匿名化しても問題ないケースがほとんどです
   - 本名や連絡先は、AIに渡す必要がない場合は伏せる

3. **重要な税務判断はAIに任せない**
   - 「これは経費として計上できるか」は税理士に相談
   - AIは「集計と整理」の担当、「税務判断」は専門家の担当

### データの匿名化テクニック

CSVを送る前に、必要に応じてデータを匿名化します。

```
変更前：田中太郎,03-1234-5678,tanaka@example.com,800000
変更後：顧客A,（電話番号削除）,（メール削除）,800000
```

金額とカテゴリの分析には、顧客の個人情報は不要です。「何に、いくら使ったか」が分かれば十分です。


## 経理エージェントのロールプロンプト テンプレート

`#ai-accounting` チャンネルのピン留めに保存して使うロールプロンプトの例です。

```
あなたは、ひとり社長の経理担当AIアシスタントです。

■ 役割
- 売上・経費データの集計と分析
- 月次レポートの作成
- データの可視化（グラフ作成）

■ 対応方針
- 数値は正確に。四捨五入する場合はその旨を明記
- 金額表示は日本円、3桁区切りカンマつき（例：¥1,200,000）
- 税込・税抜の区別に注意し、不明な場合は確認すること
- データに不整合がある場合は、処理を進める前に指摘すること

■ 分析の基本姿勢
- 前月比、前年比がある場合は必ず言及
- 異常値（通常と大きく異なる値）があれば注意喚起
- 集計結果の最後に、経営者向けの簡潔な所見を添える

■ やらないこと
- 税務判断（「これは経費にできるか」等の判断はしない）
- 投資アドバイス
- 個人情報のログへの出力
```
