## 営業の本質は「人と向き合うこと」

ひとり社長にとって、営業は避けて通れない仕事です。どんなに素晴らしいスキルを持っていても、お客様を見つけて仕事をいただかなければ、売上はゼロです。

しかし、営業活動には「人と向き合う仕事」と「書類を作る仕事」の2種類があります。

- **人と向き合う仕事**：ヒアリング、関係構築、クロージング、交渉
- **書類を作る仕事**：提案書、見積書、フォローメール、顧客リスト管理

前者は、あなたにしかできません。後者は、AIに任せられます。

`#ai-sales` チャンネルのAI営業アシスタントに、営業の「裏方仕事」を一手に引き受けてもらいます。


## AI営業アシスタントの担当業務

| 業務 | AIにできること |
|:---|:---|
| 提案書作成 | ヒアリング内容をもとにドラフトを自動生成 |
| 見積書の下書き | 条件を伝えれば、項目と金額の叩き台を作成 |
| フォローメール | 顧客ごとにカスタマイズした文面を一括生成 |
| 顧客情報整理 | 散在する顧客メモを構造化して一覧にする |
| 営業資料 | 事例紹介シート、サービス比較表の作成 |


## 顧客情報の整理と活用

営業活動では、あらゆる場面で顧客の情報が散らばります。名刺交換のメモ、商談の議事録、メールのやり取り、Excelの案件管理表。ひとり社長の場合、これらが頭の中とスマホのメモ帳とPCのあちこちに点在していることが珍しくありません。

AI営業アシスタントは、こうした散在する顧客情報を整理して構造化する作業が得意です。たとえば、箇条書きのメモを渡すだけで、以下のような顧客カルテに変換してくれます。

| 項目 | 内容 |
|:---|:---|
| 会社名 | 株式会社サンプル商事 |
| 担当者 | 営業部 鈴木部長 |
| 業種 | 食品卸売業 |
| 従業員数 | 約50名 |
| 課題 | Webサイトの老朽化、スマホ非対応 |
| 予算感 | 100万〜150万円 |
| 検討状況 | 提案待ち |
| 次のアクション | 提案書送付（4月中旬） |

散在していたメモがこの形に整理されるだけで、次に何をすべきかが明確になります。商談前の情報確認も一瞬で終わります。

顧客カルテの一覧をCSVで管理しておけば、AI経理（第7章）と組み合わせて「顧客別の売上分析」にもつなげられます。


## ハンズオン⑬ ヒアリング情報から提案書ドラフトを自動作成する

営業の要となる「提案書」をAIに作らせます。

商談でヒアリングした内容を、ざっくりとしたメモで構いませんのでAIに渡します。AIが、それを整った提案書の構成に変換してくれます。

**`#ai-sales` チャンネルで、以下のメッセージを送信してください：**

```
@Claude 以下のヒアリングメモをもとに、提案書のドラフトを作成してください。

【ヒアリングメモ】
クライアント：株式会社サンプル商事（食品卸売業）
担当：営業部 鈴木部長
課題：
- 現在のWebサイトが10年前のデザインで古い
- スマホで見づらいとお客様から言われている
- 問い合わせフォームがなく、電話対応に時間を取られている
- 採用ページもないので、求人サイトに高いお金を払っている

要望：
- モダンなデザインにリニューアルしたい
- スマホ対応は必須
- 問い合わせフォームをつけたい
- 採用ページも新設したい
- できれば自分たちでお知らせを更新したい（CMS）

予算：100万〜150万円
希望納期：3ヶ月以内

【提案書の構成】
1. 表紙（提案タイトル、日付、当社名）
2. ご挨拶・本提案の目的
3. 貴社の現状課題の整理（ヒアリング内容を元に）
4. ご提案内容（具体的な施策）
5. 制作範囲と仕様
6. スケジュール案（マイルストーン形式）
7. お見積り概算
8. 当社の実績・強み
9. 次のステップ

出力形式：
- マークダウン形式で、各セクションの内容を記述
- 見積り概算は項目と概算金額の表形式で
- スケジュールは週単位のマイルストーン表で
```

Claude Codeセッションが起動し、かなりのボリュームの提案書ドラフトが返ってきます。

### 返ってきたドラフトの活用法

AIが生成した提案書ドラフトは、あくまで「叩き台」です。以下のステップで仕上げてください。

1. **内容の確認**：ヒアリング内容が正しく反映されているか
2. **金額の調整**：概算金額は自分の単価テーブルに合わせて修正
3. **実績の差し替え**：「当社の実績」部分は、自分の実際の事例に書き換え
4. **トーンの調整**：クライアントの雰囲気に合わせて、硬い/柔らかいを調整
5. **体裁の仕上げ**：PowerPointやGoogleスライドに落とし込む

ゼロから提案書を書くと2〜3時間かかりますが、AIドラフトの修正なら30分〜1時間で済みます。浮いた時間を、提案内容の練り込みや、プレゼンの練習に充てられます。


## ハンズオン⑭ 顧客リストから個別フォローメールを一括生成する

営業で見落としがちなのが「フォローアップ」です。商談後のお礼メール、しばらく連絡が途絶えた顧客へのご無沙汰メール。やるべきだと分かっていても、つい後回しにしてしまう。

AI営業アシスタントに、顧客リストをもとにした個別フォローメールの一括生成を頼みます。

**`#ai-sales` チャンネルで、以下のメッセージを送信してください：**

```
@Claude 以下の顧客リストに対して、それぞれ個別のフォローメールを作成してください。

【顧客リスト】
1. A社・田中部長：先月コーポレートサイトを納品済み。最近アクセス数が伸びていると聞いた。
2. B社・山田社長：3ヶ月前に見積もりを提出したが、返事がない。予算の問題かも。
3. C社・佐藤さん：半年前にロゴデザインを担当。ECサイトの立ち上げを検討中とのこと。
4. D社・高橋課長：先週の交流会で名刺交換。AI活用に興味があると言っていた。

【メールの条件】
- 各顧客の状況に合わせた個別の内容にすること
- 件名は開封したくなる一文に
- 本文は200〜300文字程度（長すぎない）
- 押し売り感を出さず、相手にメリットがある情報提供を心がけて
- 署名は「山田太郎 / フリーランスWebデザイナー / taro@example.com」

【トーン】
- ビジネスライクだけど温かみがある
- 「お世話になっております」で始まる定型ではなく、少し個性的な書き出しで
```

4通のメールが、それぞれの顧客の状況に合わせてカスタマイズされて返ってきます。

A社には「アクセス数が伸びていると聞きました」という具体的な言及が入り、B社には予算面に配慮した柔らかい再提案が含まれ、C社にはECサイトに関する情報提供が盛り込まれ、D社には名刺交換のお礼とAI活用の話題が入る。

一括でありながら、一通一通が「個別に書いた」ように見えるフォローメール。これがAI営業アシスタントの真骨頂です。


:::message
**AIが書いた文章をそのまま送らない**

非常に重要なことなので、強調します。

**AIが生成したメール文面は、必ずあなた自身の目で確認してから送信してください。**

理由は3つあります。

1. **事実誤認のリスク**：AIが顧客情報を取り違えたり、存在しない事実を書いたりする可能性があります。「御社のECサイトの件で」と書いたメールが、ECサイトを持っていない会社に届いたら、信頼を失います。

2. **トーンのミスマッチ**：メールの文面は、送り手と受け手の関係性によって微妙に変わるものです。AIはその関係性の「温度感」までは完全に把握できません。

3. **あなたらしさの欠如**：AIが書いた文章は、上手ではあるものの、あなたの個性や人柄が反映されていません。最後にひと手間、自分の言葉を添えるだけで、メールの温度がぐっと変わります。

AIが作るのは「下書き」。仕上げるのは、あなたです。
:::


## 営業データの管理について

顧客情報をAIに渡す際は、第7章で触れたセキュリティの注意点を思い出してください。

- 顧客の個人情報は必要最小限にする
- 機密性の高い取引条件（特別値引き率など）は含めない
- 顧客の社内情報（未公開の経営情報など）は、AIに渡さない

営業の世界では、信頼関係がすべてです。顧客から聞いた情報の取り扱いには、AI活用であっても慎重を期してください。


## 営業で使える追加プロンプト集

### 事例紹介シートの作成

```
@Claude 以下の案件情報から、営業用の事例紹介シートを作成してください。

案件：A社コーポレートサイトリニューアル
課題：10年前のデザイン、スマホ非対応、問い合わせ導線がない
施策：レスポンシブデザイン、WordPress導入、問い合わせフォーム設置
成果：問い合わせ数が月5件→月20件に増加、ページ閲覧数150%増

形式：
- Before / After 形式
- 数値の成果を目立たせる
- 300字以内のコンパクトな構成
```

### 商談前の準備シート

```
@Claude 明日、○○株式会社（業種：○○）との初回商談があります。
商談準備シートを作成してください。

含めてほしいこと：
- 会社概要（Webで調べられる範囲で）
- 業界の最近のトレンド3つ
- 想定される課題（この業種でよくある課題）
- ヒアリングすべき質問リスト（10問）
- 当社サービスの刺さりそうなポイント
```

### 失注分析

```
@Claude 過去半年で失注した案件を分析したいです。
以下のデータから、失注の傾向と対策を考えてください。

失注案件：
1. X社：見積もり提出後3週間で「予算が合わない」と連絡
2. Y社：コンペで負け。大手制作会社に決定
3. Z社：「社内で検討する」と言われたまま音沙汰なし

分析してほしいこと：
- 失注パターンの分類
- 各パターンへの対策案
- 営業プロセスの改善提案
```
