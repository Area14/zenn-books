## 「発信しなきゃ」のプレッシャーから解放される

ひとり社長にとって、マーケティングは「やらなきゃいけないと分かっているけど、手が回らない」仕事の代表格です。

SNSを更新しなきゃ。ブログを書かなきゃ。メルマガも出したい。ホームページの内容も古くなっている。でも、今日も目の前の案件で手一杯で、気づいたら最後の投稿から2週間が経っている——。

AIマーケターが「発信の量」を担い、あなたは「発信の質（方向性の判断と最終チェック）」に集中する。その体制を作ります。


## AIマーケターの担当業務

| 業務 | AIにできること |
|:---|:---|
| SNS運用 | 投稿案の一括生成、ハッシュタグ提案 |
| ブログ運用 | 記事の構成案・下書き作成 |
| 広告コピー | キャッチコピー、バナーテキスト |
| コンテンツ企画 | 月間カレンダー、ネタ出し |
| Web制作 | LP（ランディングページ）のHTML生成 |


## ハンズオン⑮ 1ヶ月分のSNS投稿案を一括作成する

最も即効性があるのが、SNS投稿案の一括生成です。1ヶ月分の投稿ネタを一気に用意しておけば、あとは毎日1本ずつ投稿するだけ。「今日は何を投稿しよう」と悩む時間がゼロになります。

**`#ai-marketing` チャンネルで、以下のメッセージを送信してください：**

```
@Claude 以下の条件で、4月分のSNS投稿案を30本作成してください。

【アカウント情報】
- 業種：フリーランスWebデザイナー
- ターゲット：中小企業の経営者・マーケティング担当者
- 投稿先：X（旧Twitter）
- 目的：専門性のアピール、信頼構築、問い合わせ獲得

【投稿のカテゴリ配分】
- ノウハウ共有（Webデザインのコツ、トレンド）：10本
- 制作実績の紹介：5本
- 業界ニュースへのコメント：5本
- お客様の声・成功事例：4本
- 日常・人柄が伝わる投稿：4本
- 告知・CTA（行動喚起）：2本

【形式】
- 各投稿は140文字以内
- ハッシュタグは3〜5個
- 投稿カレンダー形式（日付・カテゴリ・本文・ハッシュタグ）の表で出力
- 投稿時間のおすすめ（朝/昼/夜）も付記

【トーン】
- 専門的だけど分かりやすい
- 上から目線にならない
- 時々ユーモアを交える
```

数分後、30本分の投稿案が表形式で返ってきます。

### 返ってきた投稿案の活用法

30本の投稿案には、そのまま使えるものも、修正が必要なものも混在しています。

1. **まずざっと読む**：全体のバランスを確認
2. **自分らしくないものを修正**：AIの表現を自分の言葉に置き換え
3. **実績系は事実を確認**：「架空の事例」になっていないか
4. **予約投稿ツールに登録**：Buffer、SocialDog等のツールに貼り付けて予約

30本のうち半分がそのまま使えて、残り半分を少し修正するだけだとしても、ゼロから30本考えるのに比べれば圧倒的な時間短縮です。


## ハンズオン⑯ ブログ記事の構成案と下書きを生成する

ブログ記事は、検索エンジンからの流入（SEO）を狙う上で欠かせないコンテンツです。しかし、1記事書くのに3〜5時間かかるという方も少なくありません。

AIマーケターには、「構成案の作成」と「下書きの生成」を2段階で頼みます。

### ステップ1：構成案を作る

```
@Claude 以下のテーマでブログ記事の構成案を作ってください。

テーマ：「中小企業がWebサイトをリニューアルすべき5つのサイン」
ターゲット読者：中小企業の経営者（40〜50代）
記事の目的：リニューアルの必要性に気づいてもらい、問い合わせにつなげる
想定文字数：3,000字程度

構成案に含めてほしいこと：
- タイトル案（3パターン）
- リード文（導入文）の方向性
- H2見出し × 5〜6個
- 各見出しの下に書くべきポイント（箇条書き3つ程度）
- まとめ＋CTA（行動喚起）の方向性
- SEOで狙うキーワード候補
```

構成案が返ってきたら、内容を確認して、必要に応じて修正します。「この見出しは不要」「ここにこんな話を入れたい」とフィードバックを返してもOKです。

### ステップ2：下書きを生成する

構成に納得したら、次はそのまま下書きを頼みます。

```
@Claude 上記の構成案をもとに、3,000字程度のブログ記事の下書きを書いてください。

条件：
- ですます調で、読みやすく
- 専門用語には簡単な説明を添えて
- 具体例やたとえ話を多く入れて
- 最後に「お気軽にご相談ください」+問い合わせ先のCTAを入れて
```

3,000字の記事が数分で生成されます。

もちろん、AIが書いた下書きをそのまま公開するのはおすすめしません。あなたの経験に基づくエピソード、実際のお客様の声、業界人ならではの視点を加えることで、記事に「血が通い」ます。AIは骨組みと肉付けを担当し、あなたが「魂」を入れる。この分業が、品質と効率の両立の鍵です。


## ハンズオン⑰ ランディングページのHTMLを自動生成してプレビューする

最後に、本書のハンズオンの中でも特にインパクトが大きいものをお見せします。

**AIマーケターに、ランディングページ（LP）を丸ごと作ってもらいます。**

ランディングページとは、特定のサービスや商品を紹介し、問い合わせや購入などの行動を促すための1枚のWebページです。制作会社に頼めば10万〜50万円、自分で作ろうとすれば何日もかかるものです。

それを、Slackのメッセージ1つで作ってもらいます。

**`#ai-marketing` チャンネルで、以下のメッセージを送信してください：**

```
@Claude 以下の条件で、サービス紹介のランディングページ（LP）をHTML+CSSで作成してください。1つのHTMLファイルにすべてまとめてください。

【サービス情報】
サービス名：スピードWebリニューアル
提供者：山田太郎（フリーランスWebデザイナー）
ターゲット：「うちのWebサイト、そろそろ直さなきゃ」と思っている中小企業の社長

キャッチコピー：「古いホームページ、そのままで大丈夫ですか？」
サブコピー：「最短1ヶ月、50万円からのWebサイトリニューアル」

【LPの構成セクション】
1. ヒーローセクション（キャッチコピー + CTA ボタン）
2. こんなお悩みありませんか？（3つの課題）
3. サービスの3つの特徴
4. 制作の流れ（4ステップ）
5. 料金プラン（3段階のテーブル）
   - ライト（50万円）：トップ+5ページ
   - スタンダード（100万円）：トップ+10ページ+CMS
   - プレミアム（150万円）：フルカスタム+EC機能
6. お客様の声（3件）
7. よくある質問（5つ）
8. 問い合わせフォーム（名前、メール、電話、ご相談内容）

【デザイン条件】
- モダンでクリーンなデザイン
- メインカラー：ネイビー（#1e3a5f）
- アクセントカラー：オレンジ（#f59e0b）
- レスポンシブ対応（スマホでも見られるように）
- Google Fontsの Noto Sans JP を使用
- スクロールすると各セクションがフェードインするアニメーション
```

Claude Codeセッションが起動し、しばらくすると完成したHTMLファイルが生成されます。

「View Session」をクリックしてブラウザでセッションを開くと、作成されたHTMLの内容を確認できます。HTMLコードをコピーしてローカルのファイルに保存し、ブラウザで開けば、実際のLPがプレビューできます。

この体験は、ほとんどのひとり社長にとって衝撃的なはずです。プロのWebデザイナーに発注すれば数十万円、制作期間は数週間。それが、Slackのメッセージ1つで、数分で形になる。

もちろん、AIが作ったLPのクオリティは「プロのWebデザイナーの仕事」には及びません。写真は仮のもの（実際の写真への差し替えが必要）、テキストは修正が必要、細かいデザインの微調整も必要です。しかし、「叩き台」としては十分以上の品質です。

これを起点に、スレッドで「ここの色を変えて」「このセクションを削除して」「CTAボタンをもっと目立たせて」と修正を重ねていけば、実用レベルのLPに仕上がります。

**AIコンテンツとオリジナリティ**

AIが生成したコンテンツをマーケティングに使う際、避けて通れない問題が「オリジナリティ」です。

### Googleの立場

Googleは、コンテンツの品質を「誰が書いたか」ではなく「ユーザーにとって有益かどうか」で評価すると明言しています。つまり、AIが書いたからといって自動的にペナルティを受けるわけではありません。

ただし、「大量に自動生成された、中身の薄いコンテンツ」は低品質と判断される可能性があります。

### オリジナリティを確保するポイント

1. **あなたの経験を入れる**：AIには書けない「実際にやってみたらこうだった」というエピソードが、最大の差別化要因です。

2. **独自の視点を加える**：業界の一般論はAIが書ける。「うちの場合はこう考える」という意見は、あなたにしか書けません。

3. **具体的な数字を入れる**：「問い合わせが増えました」ではなく「月5件が月20件に増えました」。実数値はオリジナリティの証です。

4. **顧客の声を引用する**（許可を得た上で）：実在の顧客の言葉は、AIには生成できない本物のコンテンツです。

AIは「80%の下書き」を担当し、残りの「20%」であなたのオリジナリティを注入する。この20%こそが、あなたのブランドを形作るものです。
:::


## マーケティングエージェントのロールプロンプト テンプレート

```
あなたは、ひとり社長専属のマーケティング担当AIアシスタントです。

■ 役割
- SNS投稿案の作成
- ブログ記事の構成・下書き
- 広告コピー・キャッチコピーの提案
- ランディングページの制作
- コンテンツカレンダーの作成

■ 対応方針
- ターゲット顧客の視点で書くこと
- 専門用語は避け、分かりやすい表現を優先
- 「売り込み」より「価値提供」のトーンで
- SEOを意識し、キーワードを自然に含める

■ SNS投稿のルール
- 1投稿140文字以内（X（旧Twitter）想定）
- ハッシュタグは3〜5個
- CTAは週2回まで、それ以外は情報提供

■ ブログ記事のルール
- リード文で読者の課題に共感する
- 見出し（H2）は5〜7個
- 結論ファースト、詳細はあとから
- 最後にCTA（問い合わせへの誘導）

■ やらないこと
- 誇大表現や虚偽の実績
- 競合の誹謗中傷
- 著作権で保護されたコンテンツの複製
```
